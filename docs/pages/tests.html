Utility - Tests

ApPHP Framework and Applications Tests.

This utility demonstrates basic features for testing applications and a framework. while the code development process. You may select the project and project components for testing: helpers in framework, controller actions in single application, etc.

Remember: this utility doesn't replace the whole testing process for development of application, but only demonstrates the basic features and the way you can use them.

By default this utility is placed in a following directory: 
utils/tests/
To access it after installation of framework, type in your browser: 
http://hostname/utils/tests/
