Running examples

After successfull installation you may run embedded examples, that demonstrate all basic features of the ApPHP Framework.

All examples are found in the folder called demos/

To run them simply type in your browser: 
http://hostname/{framework-directory}/demos/
On the demos page you may select any of available examples and run it by clicking the appropriate link.

Currently there are following demos available:
  1. Hello World
  2. Static Site
  3. Simple Login System
  4. Simple Blog
  5. Simple CMS
