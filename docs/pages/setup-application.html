Development - Setup Application

A new installation of application, based on ApPHP MVC Framework is a very straight forward process:
  • Step 1. Decompressing Downloaded Archive
  • Step 2. Uploading Files
  • Step 3. Creating Database and User
  • Step 4. Running Setup Wizard

Step 1. Uncompressing downloaded file. ¶

Decompress the application script archive on your machine.


Step 2. Uploading Files. ¶

Upload this directory (all files and sub-directories it includes) to your document root (public_html, www, httpdocs, etc.) or to your website directory using FTP.

Pay attention: DON'T use the capital letters in the name of the folder (for Linux users). For example: 
public_html/
or 
public_html/{application directory}/


Step 3. Creating Database and User. ¶

Using your hosting Control Panel, phpMyAdmin or another tool, create new database and user, and assign that user to the database. Write down the database host, name of the database, username and password. Grant this user all required privileges on that database to perform installation procedure.


Step 4. Running Setup Wizard. ¶

Now you can run setup wizard. To do this, open your Internet browser and type in Address Bar following: 
http://{www.example.com}/
or 
http://{www.example.com}/{application directory}/
If the script still not installed you will see the setup wizard starting page. Follow the instructions on the screen. You will be asked to enter: database host, database name, username and password. Also you have to enter admin email, username and password, that will be used to access the administration area of the site. After successful installation you will see an appropriate message. In the case of errors, setup wizard will display them on the screen.

While running setup wizard you could select whether to install a fresh copy of the script or just update it.
