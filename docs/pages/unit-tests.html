To install and run Unit test you have to perform following actions:

  1. Make sure you added following to composer.json file: 
    "scripts": {
    "tests-result": "phpunit --colors=always --log-junit test-results.xml",
    "tests": "phpunit --colors=always",
    "test": "phpunit --colors=always --filter"
}
  2. Install composer on your server. https://getcomposer.org

  3. After Composer is installed, install PHPUnit by
    composer require phpunit/phpunit --dev


    you may also remove it and re-install:
    composer remove phpunit/phpunit --dev
composer update
